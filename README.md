## 로컬 실행 방법

Container

- `cd ecomm/container`
- `npm install`
- `npm run start`
- http://localhost:8080/

Products

- `cd ecomm/products`
- `npm install`
- `npm run start`
- http://localhost:8081/

Cart

- `cd ecomm/cart`
- `npm install`
- `npm run start`
- http://localhost:8082/
